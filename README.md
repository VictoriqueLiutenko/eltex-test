# eltex-test

Githib pages: https://victoriqueliutenko.github.io/eltex-test/dist/

### Install

Git clone:
```bash
git clone git@github.com:VictoriqueLiutenko/eltex-test.git
```
Install dependencies:
```bash
npm i
```

Build:
```bash
gulp
```

Watch:
```bash
gulp watch
```

Serve:
```
cd dist/
php -S localhost:9000
```

Open: http://localhost:9000
